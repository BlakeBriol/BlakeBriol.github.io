Work-Hours Assistant

This project was created so that I could more easily input work-hours, track trends, and manage a large data set of entries. This has 9 features including:

  • Data Input (Job, Date, Clock-in, Clock-out, and Wage)
  • Dataframe Print
  • Entry Deletion
  • Database Clearing
  • Total Revenue
  • Taxed Revenue
  • Graph Trends
  • Give Statistics
  • Database Backup (Automatic with Data Input)

I am currently working towards this project being a fully isolated program.
